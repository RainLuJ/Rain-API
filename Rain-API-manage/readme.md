# 接口管理模块

## 用户管理

1. 用户信息的增删改查
2. 普通的账号密码注册用户 & 使用邮箱注册用户
3. 注册用户时给用户分配accessKey & secretKey，并提供重新获取accessKey & secretKey的接口



## 接口管理

1. 接口信息的增删改查
2. 上线接口 & 下线接口
3. 反射接口调用（反射获取接口对应的SDK-Client，通过反射调用Client中的方法实现更加灵活、解耦的接口调用方式）
4. 提供下载本项目中所管理的接口所对应的SDK接口


## 用户-接口关系管理

1. 用户-接口信息的增删改查
2. 获取免费的接口调用次数


## 数据分析

1. 统计调用次数最多的接口信息（TOP）
2. 统计购买次数最多的接口信息（TOP）
